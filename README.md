# exteraGram Landing Concept

A fan-made, high-performance landing page concept for [exteraGram](https://exteragram.app). 

This project was built as a frontend challenge to demonstrate proficiency in modern web technologies, specifically focusing on **Canvas API** for interactive backgrounds and **Glassmorphism** UI design.

> **Note:** This is a fan-made project created for educational and portfolio purposes. It is not affiliated with the official exteraGram team.

## 🛠 Technical Highlights
This project showcases the following skills:
*   **Canvas API:** Custom implementation of particle physics and mouse-interaction logic.
*   **Performance Optimization:** Efficient animation loops using `requestAnimationFrame`.
*   **Responsive Web Design:** Fluid layouts using CSS Flexbox and Grid.
*   **UI/UX:** Modern glassmorphism aesthetic with CSS backdrop-filters and smooth transitions.
*   **Zero Dependencies:** Built with pure HTML5, CSS3, and Vanilla JavaScript (no frameworks used).
